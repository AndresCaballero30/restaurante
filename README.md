# Proyecto de Gestión de Restaurantes

## Descripción del Proyecto
Este proyecto es una aplicación web full-stack diseñada para la gestión integral de restaurantes. Incluye módulos para la administración de menús, gestión de pedidos, control de inventario y organización de mesas. Además, cuenta con un sistema robusto de autenticación de usuarios y dashboards analíticos para una toma de decisiones informada.

## Tecnologías Utilizadas

### Backend
*   **Runtime:** Node.js
*   **Framework:** Express.js
*   **Base de Datos:** SQLite (gestionada a través de `better-sqlite3`)
*   **Autenticación:** JWT (JSON Web Tokens) para la gestión de sesiones y Bcrypt para el hash seguro de contraseñas.
*   **ORM/Acceso a Datos:** SQL Directo (interacción con la base de datos mediante consultas SQL explícitas).

### Frontend
*   **Framework:** React
*   **Herramienta de Construcción:** Vite
*   **Librería UI:** shadcn/ui (construido sobre Radix UI y estilizado con Tailwind CSS)
*   **Enrutamiento:** React Router DOM
*   **Gestión de Formularios:** React Hook Form con Zod para validación de esquemas.
*   **Obtención de Datos/Gestión de Estado:** TanStack Query
